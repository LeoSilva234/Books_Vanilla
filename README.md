# Books_Vanilla

Fork do projeto integrador Full Stack Java 2024.2 (curso), usado como referência de estudo de uma API REST completa em Spring Boot.

Stack: Java, Spring Boot (Web, Data JPA), banco H2/SQL, upload de arquivos.

O que pratiquei: leitura de uma API em camadas (controller, model, repository), autenticação com criptografia de senha (PasswordEncryptor) e fluxo de CRUD de livros e contas de usuário.
